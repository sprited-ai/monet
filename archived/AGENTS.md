# Working conventions for the Monet repo

## BACKLOG discipline (always)

- `BACKLOG.md` at the repo root is the single source of truth for outstanding work.
- **Capture:** whenever TODO work appears — from Jin or discovered mid-task — add it to
  `BACKLOG.md` immediately, in the right section.
- **Cross off:** when a task is finished, update `BACKLOG.md` in the same turn — move it to
  `## Done` (or mark `[x]`). Never leave completed work sitting in the active list.
- **Keep current:** reconcile `BACKLOG.md` at the end of every working session so it always
  reflects reality. If a task is blocked or owned by Jin, note that next to it.
