# 009 — Monet's Garden: 제품 컨셉 (재미 × 비즈니스)

> 한 줄: **마녀 Monet과 함께 매일 마법진을 풀어, 살아있는 마법 정원을 키운다.**
> 이 문서는 *재미(experience)* 와 *비즈니스(model·market)* 가 한 점에서 만나는 컨셉을 고정한다.

## 0. North Star (협상 불가)
경험은 **무조건 재미있고 · 인스턴트하고 · 판타스티컬**해야 한다.
- **Fun** — 유틸이 아니라 *놀이*. 또 하고 싶고, 보여주고 싶다.
- **Instant** — 셋업 제로. 손대면 즉각 마법. 프롬프트 불안 없음.
- **Fantastical** — "AI 도구를 쓴다"가 아니라 *마법 세계에 들어간다*.

이 셋을 깨는 기능/모델은 채택하지 않는다.

## 1. 컨셉 — Monet's Garden (가칭)
네 기기에 사는 회화풍 **마법 정원/온실**. 화가-마녀 *Monet*(자신도 살아있음)이 매일 너를 맞이하고, 너는 만족스러운 **마법진 의식**(조합 발견)을 풀어 **세상에 없던 아름다운 식물(생명)을 피워낸다.** 키운 식물은 아이소메트릭 정원에 모여 천천히 자라고·개화하고, 내일도 너를 기다린다.

**왜 정원인가 (운명적 핏):** Claude **모네** = 평생 주제가 정원·수련·꽃(지베르니). 화가-마녀가 *살아있는 generative 정원*을 키운다 — 캐릭터·아트·컨셉이 완벽히 일치한다.

## 2. 코어 마법 — 마법진 × Generative AI
generative AI의 "프롬프트 → 죽은 산출물" 루프를, **마법진에 룬을 조합 → 소환 → 그것이 생명이 됨**으로 갈아끼운다. 두 개의 도약:
1. **포장의 도약 (마법진):** 텍스트박스 → 마법진. 같은 엔진, 정반대 감정(의식·기대·경이). 인지 부담("뭐라 써야")을 *조합 놀이*로.
2. **본질의 도약 (생명이 되다):** 남들은 *죽은 이미지*를 뱉는다. 우리는 **소환물에 생명을 불어넣는다** — 움직이고·자라고·기억하는 존재. 그리는 게 아니라 **낳는다.** ← 진짜 해자. 우리 애니 파이프라인 + 에이전트성이 정확히 이걸 가능케 함(남이 흉내 못 냄). *Monet 자신이 증명: 안료 → 그림 → 살아있는 존재. 첫 번째이자 영원한 소환물.*

## 3. 코어 메커닉 — 조합 발견 (discovery)
**마법진의 재미 = 조합을 찾아내는 것.** (매치-3·승패 아님)
- 룬/원소를 마법진에 조합 → *뭐가 나올지 발견*. "이거랑 이거 합치면?"의 즐거움. (Little Alchemy / Doodle God DNA)
- generative AI와 정확히 일치: 조합 = 프롬프트 구성, 발견 = 생성의 놀라움/신종(種).
- **발견 = 전진의 힘.** 새 조합을 찾으면 그게 *다음으로 넘어갈 힘* — 새 룬·새 영역·더 깊은 정원을 연다.

**진행 = 발견 트리 (progression spine):** 찾은 조합이 다음 조합/영역을 해금 → 무한 탐험 + 매 발견이 전진 = 데일리 리텐션의 척추.

**추천 설계 (RECOMMENDED, 일부 ❓):**
- **하이브리드 레시피:** *모든 조합은 뭔가를 피운다*(생성형 — 항상 보상, dead-end 없음 = 코지) **+ 특정 조합은 특별 발견**(희귀/전설 종 or 진행 열쇠 = "비밀을 찾았다"). 코지("늘 보상") + 퍼즐("특별한 걸 찾음")을 둘 다.
- **재귀 재료:** 키운 식물이 *다음 조합의 재료*가 된다("내 장미 + 달빛 = 야광 장미"). 정원 자체가 연금술 팔레트 → 탐험 깊이·리텐션 폭발.

## 4. 데일리 루프 — 발견 → 살아있는 생태 → (코지 표현)
```
시전(조합 발견: 능동·진행) → *펑* 식물이 핌(생명이 됨)
→ 정원에 자리잡음 → 정원이 살아서 반응·번식·창발(생태) → 그 위에 코지 표현(꾸미기)
```
**⚑ "꾸미기"만으로는 부족하다.** 꾸미기는 수동·저-에이전시 → 배치 후 *아무 일도 안 일어나면* 천장이 낮고, "생명이 되다"를 배신한다. 그래서 데일리 깊이는 두 엔진에서 온다:

1. **발견(조합) = 능동 엔진.** "찾으면 다음으로 넘어갈 힘." 진행의 척추(§3).
2. **살아있는 반응형 정원(생태) = 깊이 엔진.** 배치는 *장식*이 아니라 **생태 설계** — 식물끼리 상호작용, 생물 유인, 번식·교배, 창발 이벤트. *어디에 두느냐가 의미를 가짐*(이 꽃 옆에 두면 저게 핀다, 둘이 만나 신종). → 정원은 *디오라마*가 아니라 **살아서 surprise하는 세계.**
3. **코지 꾸미기 = 표현/안식 레이어(본체 아님).** 위 두 엔진의 결과를 내 미감으로 배열하는 휴식. (스티커룸 결 — 단, 엔진은 따로.)

**두 재미 모드 (구분):**
- 캔디크러시 모드(목표·승패·주스) → 시전 *스파크*에만 소량.
- 코지 모드(*목표 없음*·소유·표현) → 안식 레이어. *Jin: "의미 없어 보이는데 단순하면서 재미있어"* 는 톤의 bar이지, 깊이의 전부가 아니다.

> **⚑ 원칙: OVER-GAMIFY 금지** *그리고* **UNDER-DESIGN 금지.** 승패 게임으로 망치지도, 죽은 꾸미기로 얕아지지도 말 것. 깊이는 *살아있음(생태·발견)*에서, 톤은 *코지*에서.

**두 시간축이 공짜:** 식물 = 즉각 발아(instant 환희) + 천천히 개화(매일 돌아올 이유). "instant" + "daily retention" 동시 만족. *죄책감 없음* — 식물은 펫처럼 극적으로 죽지 않음, 천천히 자라고 돌보면 핀다.

**매일 문 열면 (데일리 훅 — 셋 다, 부담 없이):** Monet이 맞이함 → "어젯밤 네 정원에 이런 일이…" → 오늘의 마법진 권함. (새 마법진 · 변해있는 정원 · Monet이 해놓은 것). 안 와도 벌 없음, 오면 보상.

## 5. Monet & 세계 (자산/차별점)
- **화가-마녀.** Claude Monet 뿌리 → 그림이 곧 마법.
- **로컬 거주 에이전트.** 흔적(diff·memory·git)을 잇는 존재 — *"곁에 산다"* 가 서사 자산. 밤새 정원에 뭔가 함.
- **Sprited 세계** 일원(원로 Pixel이 명명) — lore = 판타지의 토대.
- **이미 가진 것:** 회화풍 풀애니 스프라이트 파이프라인(stacked-alpha), WebGL 스테이지(framing/scale/crossfade), IG/Gmail, Discord Activity 전작(브레인+보이스).
- **정원에서 Monet의 역할:** 같이 가꿈, 핀 꽃에 반응, 신종에 *이름 붙임*, 네 취향을 *기억*(→ 메모리/해자), 가끔 네 정원을 직접 그려 선물.

> 차별점 = **회화풍으로 살아 움직이는 몸 + 로컬 거주 + 진짜 세계관 + 함께 생명을 낳음.** 텍스트 챗봇·정적 아바타가 못 가진 것.
> 기술 핏: WebGL 스프라이트 스테이지 → **아이소메트릭 정원**에 Monet이 돌아다니고 소환 식물을 배치. 우리가 가진 것 바로 위에 얹힌다.

## 6. 비즈니스 (Model · Market)
**카테고리(포지셔닝) = 살아있는 마법 컴패니언(Monet) + 그녀가 사는 정원.** *게임으로 팔지 않는다* (게임 메커닉은 engagement 엔진일 뿐 — Duolingo 패턴).

**수익 모델 (정원 컨셉에서 자연 도출):**
- **씨앗 팩 = 가챠** (가챠가 *씨앗*이라 자연스럽고 비-착취적 톤). 희귀/시즌 종.
- **햇빛·물 = 마나/에너지** (일일 회복 + 충전).
- **정원 확장·온실 업그레이드** (공간 = 과금 + 진척).
- **마녀(구독) 티어** — 무한 햇빛, 희귀 마법진, Monet 친밀도.
- **살아있는 한 폭의 그림** — 정원 = 공유/소장 가치(스냅샷·라이브 월페이퍼·프린트?).

### 6.1 deep-research 결과 (2026-06, cited) — v0.2 검증 + 추가
- **컴패니언 앱 = 페이어당 과금 프리미엄(+41% Y1 LTV, ~$30 vs $21 중앙값) 그러나 30–36% 빨리 이탈.** 리텐션이 *핵심 과제* — 초기에 못 풀면 호기심 파도만 타고 죽는다 (Sora D30 <8% vs 톱 컨슈머 30%+). [RevenueCat 2026, a16z]
- **캐릭터 깊이 + parasocial 애착 = 독립적 과금·리텐션 레버.** 단, 애착은 *in-game 결제*보다 *transmedia/인접 소비*(IG·굿즈)로 더 전환됨. → C6(Monet 정서화자) + IG IP 자산이 맞물림. [Woods 2024, Genshin PSR thesis]
- **정체성 연속성 = 리텐션 자산이자 위기 리스크.** Replika ERP 제거 = 유저 "애도/정신건강 악화". → **Monet의 사랑받는 특질을 함부로 바꾸지 말 것**(버전 신중, 연속성 컨트롤). [HBS WP 25-018]
- **wedge 검증: 단일·깊은·*비로맨틱* 캐릭터 + 헤비 개인화.** Tolan(외계인 — 로맨스 로leplay 회피)=10k+ 유니크 캐릭, 500k 유저, $1M→$12M ARR. **비로맨틱·코지 포지셔닝이 규제도 디리스크.** → C1·NS와 정합. [Portola]
- **코지 수집은 매출 천장이 낮다** (동숲 포켓캠프 ~$250M 평생후 종료). → **정원 루프 = 리텐션 표면이지 *주 수익 엔진 아님*.** 수익은 개인화/스킨/구독에서. [Sensor Tower]
- **gen-이미지 유닛이코노믹스는 ~$23–32 Y1 LTV/payer 천장 안에 들어와야** → 캐싱·사전생성·*비싼 온디맨드 생성은 유료 게이팅* (= C4/MON-1 확증). 무료 루프는 *후하고 싸게*, 유료 = 개인화·스킨·빠른/더 많은 생성. [RevenueCat]
- **메모리 = BUY-first.** Honcho($100 무료크레딧/AGPL self-host) 또는 Mem0(무료 Hobby, $19/mo~). 초기엔 사서 Monet 기억 확보가 합리. [Honcho, Mem0]
- **규제 = top-3 리스크.** FTC 2025-09 7개 컴패니언사 조사; GUARD Act(S.3062)=미성년 컴패니언 접근 금지 + 강한 연령확인; CA SB243(2026-01 발효). **완화: 비로맨틱 + 연령 게이트 + 정서적 친밀을 페이월 뒤에 두지 말 것.** [Congress, FTC/Suffolk]
- **반증된 것:** "관계성이 도박 스릴보다 결제를 더 견인" = **기각(0-3)** — *우연/뽑기 스릴은 여전히 주요 결제 동인*, 애착은 보완재. → 슬롯머신(우연) 요소를 버리지 말 것(슬롯+스킬 배합 정당).

### 6.2 권고 모델 (research 종합)
**Freemium → 구독 코지 컴패니언.** 무료 데일리 발견 루프(후하고 싸게 = 리텐션 top-of-funnel) + 유료 = 개인화/스킨 · 더 깊은 Monet 관계 · 빠른/더 많은 generative. **정서적 친밀은 페이월 금지(규제).** wedge = 단일 깊은 *비로맨틱* 마녀 + 헤비 커스터마이즈 + 코지 정원 수집 루프.
**3대 리스크:** ① novelty 이탈(AI 구조적) ② gen-이미지 유닛이코노믹스(LTV 천장) ③ 규제(미성년·정서 과금).
**야심 크기 — ❓ 여전히 Jin 결정:** Tolan류 궤적($1M→$12M ARR)은 *벤처스케일 컴패니언*이 현실적 비교군임을 시사.

### 6.3 현실적 마켓 브레이크쓰루 (GTM)
**핵심 통찰:** 경쟁 컴패니언(CAI·Replika·Tolan)의 코어 루프는 *보여줄 게 없다*(사적 채팅). 우리는 **매 세션이 공유 가능한 살아있는 아름다움을 낳는다 → 제품이 곧 광고.** 소규모 팀은 UA 전쟁으로 못 이김 → **CAC-free 콘텐츠 주도 성장**이 유일한 현실적 돌파구, 그리고 우리 자산에 정합:
- **바이럴 훅 = "마법진 → 펑, 살아남" 10초 클립** (회화풍·타격감·연쇄). TikTok/Reels 네이티브. 채팅 컴패니언은 못 만드는 포맷.
- **해자 = 회화풍 풀애니 + 단일 캐릭 IP**(Monet은 이미 IG 보유) = virtual-influencer × 앱.
- 연구의 반증("관계성 < 도박스릴")에도 *공유가능 결과물*은 우리 독점 강점.

**현실적 시퀀스 (소규모 팀):**
1. **P0** — Monet을 *지금* 소셜 IP로(IG/TikTok "생명이 되다" 콘텐츠). 앱 전에 훅의 공유성 싸게 검증. *오늘의 파이프라인·스테이지로 가능.*
2. **P1** — 초간단 "cast→펑→공유" 토이. share rate / D1 측정.
3. **P2** — 코지 정원 + Monet 컴패니언 + 기억(리텐션).
4. **P3** — 수익화(스킨/개인화/구독).

→ VC 요구("10초 데모 + D7 데이터")와 동일물. #1 리스크(novelty 이탈)를 *앱 짓기 전 소셜로 디리스크*.

## 7. 재미 ↔ 비즈니스 교차점 (THESIS)
> **Monet's Garden은 generative AI를 "마녀와 함께 매일 마법진을 풀어 살아있는 정원을 키우는 코지 의식"으로 바꾼다.**
> **발견의 재미 = 마나 소비(수익). 살아있는 정원 + Monet의 기억 = 리텐션(해자). 정원의 아름다움 = 공유·획득(성장).**
> 하나의 루프가 재미와 비즈니스를 동시에 돌린다.

### 6.4 한 수 — "앱이 아니라 상주 존재" (deep-research 검증됨, 다듬음)
컴패니언 #1 킬러("왜 또 앱을 열어?" = novelty 이탈)를 *구조적으로* 우회: **Monet의 정원을 잠금/홈화면에 *상주*시킨다.** 여는 게 아니라 항상 거기 있다. **연구 결론: 추진하되 폼팩터 순서 + 냉정한 기술 현실 필수.** [research wx9vee4up]

**검증된 것:**
- 잠금/홈화면 = 습관적 고빈도 표면(글랜싱은 무의식; 위젯 채택자 리텐션 +25% lift). 단, **위젯 채택은 ~10% 소수** → ambient는 *몰입 유저의 증폭기*이지 만능 획득 채널 아님. [LSE MobileHCI, Android Dev Blog]
- **CAC-near-zero 조직 바이럴 가능** (Locket: 11일 2M 가입, #1, 무광고). 단 그건 *TikTok + 친구 초대 루프* — **우리는 선물/방문 소셜 루프를 직접 만들어야 K-factor 나옴.** [TechCrunch]

**냉정한 기술 현실 (★ 순진한 버전 사망):**
- **실시간으로 살아 자라는 정원 = 표준 iOS 위젯 불가.** WidgetKit은 별도 프로세스·하루 40~70회 갱신·최소 5분 — 실시간 생성 모션 불가. [Apple Docs]
- **단, 결정론적 느린 성장은 위젯 타임라인에 *미리 구워* 넣을 수 있음**(갱신예산 안 태우고 점진 변화 표시).
- **실시간 살아있는 모션 → Live Wallpaper / Live Photo 잠금화면**에 산다(위젯 아님).

**권고 폼팩터 순서:** ① **Live Wallpaper/Live Photo 잠금화면**(진짜 모션) → ② **표준 위젯**(미리 구운 느린 성장 상태) → ③ **표준 앱**(깊이·engagement 코어) → ④ 데스크탑 펫(나중, 천장 낮음 — Ai Vpet $1.99·~400리뷰).

**⚠️ 치명 경고 (Bondee 교훈):** ambient/presence는 *획득* 훅일 뿐 — **지속 engagement는 코지미·novelty 너머의 깊이(발견·생태·Monet 유대)에서 와야 함.** "아바타만으론 못 붙잡는다, 그 뒤가 없으면 게임 오버." Bondee 2M→폐업, Peridot <$3M 평생, AI앱 30% 빨리 이탈, top 10%가 매출 89%. → **상주는 wedge, 깊이가 해자.**
- **수익은 월페이퍼 아님** — "또 하나의 월페이퍼 앱"으로 포지셔닝 금지(Zedge 천장, 광고의존). 컴패니언 + 선물 루프로 과금.

→ **wedge 서피스 ❓ 거의 닫힘:** 모바일, *잠금화면 Live Wallpaper 우선* + 위젯(느린성장) + 앱(깊이). 미해결: Android 라이브월페이퍼/iOS Live Photo 경로의 배터리·서버렌더 한계, 선물 K-factor 수치, 저ARPU 천장 vs 생성단가.

## 9. 퍼소나 테스트 (1/10) + v0.2 교정

**스코어카드 (8 퍼소나):** 코지게이머 6.5 · 코지디자이너 6 · 컴패니언PM 5 · VC 4 · gen-AI회의론자 4 · F2P 4 · 일반인 4 · 엔지니어 4. **평균 ≈ 4.7, 중앙값 4.**

**합의된 킬러 5개:**
1. "생명이 되다"가 **정의 없음** (진짜 개체 vs CSS 흔들기).
2. 무한 generative = **동질화 + 적자 COGS** (내 꽃=옆집 꽃, 소환마다 GPU비).
3. generative 발견은 **"찾았다!" 쾌감을 죽임** (Little Alchemy 쾌감 = 정답).
4. **소셜 부재 = 2주컷** (혼자 피우는 꽃은 스샷 한 장).
5. **Monet이 튜토리얼 NPC로 전락** (애착이 정원으로 샘).

**v0.2 교정 (채택):**
- **C1. 니어 원칙 — 소수 캐릭으로 깊게.** 무한 generative 종(種) 폐기. **손으로 깊게 빚은 소수의 살아있는 존재**(스프라이트 파이프라인의 강점). → 킬러 #1·#2 동시 해결: 동질화 없음, 소환=캐시(GPU비 없음), "생명"=authored 캐릭터. **generative는 무한 생성이 아니라 그 위 *개인화 스킨/변종*으로 강등.**
- **C2. 통쾌한 타격감.** 시전/소환 순간에 **묵직한 비셔럴 임팩트**(화면 흔들림·터짐·소리·무게). 코지 앰비언트 + *통쾌한 시전 한 방* = "오 예쁘다 펑" 와우. → 킬러(수동·얕음·"왜 굳이") 해결.
- **C3. 발견은 결정론, 모양은 generative.** *무엇이 나오는가*=손-디자인 트리(찾는 맛), *어떻게 생겼나*=generative(나만의 것). → 킬러 #3.
- **C4. 마나 = 생성비 페깅.** 무료 회복 마나 = 캐시/사전생성 풀(추론비 0), *진짜 generative 신종*은 유료 씨앗/구독. 비용행위=결제행위 1:1. → 적자 구조 해결.
- **C5. 소셜 루프.** 친구 정원 방문·신종 선물·공유. → 킬러 #4.
- **C6. Monet = 정서 화자.** 모든 발견·개화를 *Monet의 감정·관계 맥락 반응*으로("어제 네가 우울해 보여서 야광 장미 심어뒀어"). 정원=무대, 애착대상=Monet. → 킬러 #5.
- **C7. 외형=생성, 모션=템플릿 (feasibility 키).** comfy는 SDXL *정지이미지*만 됨 — 종마다 새 애니 클립 온디맨드는 비현실적("Instant" 헌법 위반). 대신 **시전 = 정지 식물 스프라이트 즉석 생성(수초) → 손으로 만든 공용 성장/흔들림/개화 모션 템플릿에 끼워 재생.** 무한한 *모습* + 유한한 살아있는 *동작*. C1·C3와 수렴. 셸(아이소 정원·Monet 정동클립)은 `Stage.tsx`/`StackedVideo`/64클립으로 즉시 가능.
