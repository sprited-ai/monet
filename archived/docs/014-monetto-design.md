# 014 — Monet Console

모네 콘솔은, 모넷의 인스타 포스트를 자동으로 만들어 주는 에이전트인데, 이건 나만 쓸 수 있게 한다.

먼저 가장 필요한 것은 비디오를 만들수 있어야 하는데, 내가 monet-console.sprited.ai에 들어가서 

"모넷 인스타그램 포스트 만들어줘" 하면

적당한 스타팅 이미지를 찾고 

comfy.sprited.ai에 프롬프트를 보내서 영상을 만들고, 그걸 역어서 하나의 시퀀스로 만들어 주고,

그걸 웹사이트에서 바로 보게 해줘.

그게 첫 프로토타입이야.

컨버세이셔널 에이전트라는게 좀 스콥이 커짐.

차라리, 컨버세이셔널 에이전트는 워크플로우를 만들고

그걸 유아이에 프리젠트 하고 나에게 검토하게 하고

그렇고 나서 내가 generate버튼을 누르게 하고 결과값을 내가 볼 수있게 해주고 그런게 낳을것 같은데...
