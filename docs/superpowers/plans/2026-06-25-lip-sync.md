# Lip-sync Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make Monet's mouth open and close in sync with her live TTS speech — an amplitude-driven procedural mouth composited frame-exact on her face, built so viseme sprites drop into the same slot later.

**Architecture:** Three swappable units + a tuning harness. (1) `voice.ts` taps the TTS Web-Audio graph with an `AnalyserNode` and exports `mouthOpen(): number` (0..1, 0 when silent). (2) `CharacterNode` reads an injected `mouthOpenSource` provider each frame and uploads it as the `uMouthOpen` uniform. (3) `sprite.frag` draws a procedural open-mouth in the already-erased mouth region. A new `/mouth` lab route drives the **real** `CharacterNode` (no duplicated render path) so what you tune in the lab is exactly what ships.

**Tech Stack:** TypeScript, React, WebGL2 (GLSL ES 3.00), WebCodecs (`StreamingClip`), Web Audio (`AnalyserNode`), gl-matrix, Vite.

## Global Constraints

- All Monet clips are **24 fps**, stacked-alpha mp4 (640×1280: color top 640, alpha-luma bottom). `FPS = 24`.
- The mouth box / polygon / skin live in **clip-frame u-space** (0..1 over the color top-half), the same space `u` has inside `sprite.frag`'s `erase()`.
- Mouth must be rendered **inside the billboard** (in `sprite.frag`), never as a 2D canvas overlay — the whiteroom is a 3D scene and an overlay would drift under camera moves.
- `mouthOpen()` MUST return **0 when nothing is playing** (closed mouth = no regression from today's mouthless baseline).
- Source of truth for the mouth shader is **`sprite.frag`** only. Do not fork the mouth GLSL into `WebCodecsStage.tsx`'s inline shader.
- No unit-test runner exists (tests = Playwright e2e only). Verification gate per task = **`npm run build` compiles** + **visual check at `/mouth`** (and `/` for the live task).
- Routing is a `window.location.pathname` switch in `src/App.tsx`. Dev server: `npm run dev` (from `ui/`).
- Commit straight to `main` (this repo's workflow). End commit messages with the Co-Authored-By trailer.

- [ ] **Step 2: Write `MouthLab.tsx` — decode one clip through a real CharacterNode**

```tsx
import { useEffect, useRef } from 'react'
import { mat4, vec3 } from 'gl-matrix'
import { CharacterNode } from './scene/nodes/CharacterNode'
import type { Frame, Framing, Mouth, Toggles } from './scene/types'
import { webCodecsSupported } from './webcodecs/ClipDecoder'

// /mouth — isolated lab for tuning Monet's lip-sync mouth. It drives the REAL CharacterNode
// (same sprite.frag/vert that ships in the whiteroom) with a flat front camera, so the mouth
// shape tuned here is exactly what renders live. Later tasks add a uMouthOpen slider + audio
// drivers. See docs/superpowers/specs/2026-06-25-lip-sync-design.md.

const CLIP = 'monet-talk-2' // a talking clip that has a .mouth.json sidecar
const FRAMING: Framing = { frame: [640, 640] } // color frame is 640×640; origin/scale default

const fetchMouth = (name: string): Promise<Mouth | null> =>
  fetch(`/contents/monet/${name}.mouth.json`).then((r) => (r.ok ? r.json() : null)).catch(() => null)

export default function MouthLab() {
  const canvasRef = useRef<HTMLCanvasElement>(null)

  useEffect(() => {
    if (!webCodecsSupported()) return
    const cv = canvasRef.current!
    const gl = cv.getContext('webgl2', { premultipliedAlpha: false, alpha: false })
    if (!gl) return

    const char = new CharacterNode(gl, cv.parentElement!)
    char.setClip(`/contents/monet/${CLIP}.mp4`, FRAMING, null, fetchMouth(CLIP))

    const view = mat4.create()
    const proj = mat4.create()
    const right = vec3.fromValues(1, 0, 0)
    const ambient = vec3.fromValues(1, 1, 1)

    let raf = 0
    let last = 0
    const loop = (now: number) => {
      const dt = last ? now - last : 16
      last = now
      const w = cv.clientWidth, h = cv.clientHeight
      if (cv.width !== w) cv.width = w
      if (cv.height !== h) cv.height = h
      gl.viewport(0, 0, cv.width, cv.height)
      gl.clearColor(0.93, 0.92, 0.9, 1)
      gl.clear(gl.COLOR_BUFFER_BIT)

      // Flat front view: camera looks at her mid-body; perspective is fine (the billboard
      // turns to face the camera via `right`, so there's no foreshortening of the sprite).
      mat4.lookAt(view, [0, 1.1, 4], [0, 1.1, 0], [0, 1, 0])
      mat4.perspective(proj, 0.6, cv.width / cv.height, 0.1, 100)

      const frame: Frame = {
        gl, now, dt, view, proj, right, zoom: 1, ambient,
        width: cv.width, height: cv.height, toggles: {} as Toggles,
      }
      char.update(frame)
      char.draw(frame)
      raf = requestAnimationFrame(loop)
    }
    raf = requestAnimationFrame(loop)

    return () => {
      cancelAnimationFrame(raf)
      char.dispose()
    }
  }, [])

  return (
    <div style={{ position: 'fixed', inset: 0, background: '#eeece8' }}>
      <canvas ref={canvasRef} style={{ width: '100%', height: '100%', display: 'block' }} />
    </div>
  )
}
```

```glsl
// Procedural open-mouth, drawn into the already-erased mouth region. `box` is the mouth
// AABB in u-space; `open` in 0..1. v0 = a dark ellipse whose vertical radius grows with
// `open`; a small floor keeps a thin closed line at rest. Viseme sprites replace this later.
vec3 mouthDraw(vec3 rgb, vec2 u, vec4 box, float has, float open) {
  if (has < 0.5) return rgb;
  vec2 c = vec2((box.x + box.z) * 0.5, (box.y + box.w) * 0.5);
  float w = box.z - box.x;
  float rx = w * 0.42;
  float ry = max(w * 0.04, w * 0.34 * open);
  vec2 d = (u - c) / vec2(rx, ry);
  float e = length(d);                 // <1 inside the ellipse
  float inside = 1.0 - smoothstep(0.88, 1.12, e);
  vec3 mouthColor = vec3(0.18, 0.07, 0.09); // dark interior, slightly warm
  return mix(rgb, mouthColor, inside);
}
```

- [ ] **Step 1: Add the analyser + `mouthOpen()` to `voice.ts`**

Add module state near the top (after `let curSrc`):

```ts
let analyser: AnalyserNode | null = null
let tdBuf: Float32Array | null = null
let openSmoothed = 0

function getAnalyser(c: AudioContext): AnalyserNode {
  if (!analyser) {
    analyser = c.createAnalyser()
    analyser.fftSize = 1024
    tdBuf = new Float32Array(analyser.fftSize)
    analyser.connect(c.destination) // analyser → speakers, connected once
  }
  return analyser
}

// Current lip-sync openness (0..1) from the TTS audio being played. 0 when silent or no
// Web Audio. Called once per render frame by CharacterNode.mouthOpenSource. EMA-smoothed,
// opening faster than it closes so speech reads natural. See the lip-sync design doc.
export function mouthOpen(): number {
  if (!analyser || !tdBuf) return 0
  analyser.getFloatTimeDomainData(tdBuf)
  let sum = 0
  for (let i = 0; i < tdBuf.length; i++) sum += tdBuf[i] * tdBuf[i]
  const rms = Math.sqrt(sum / tdBuf.length)
  const FLOOR = 0.01, RANGE = 0.18
  const target = Math.min(1, Math.max(0, (rms - FLOOR) / RANGE))
  const k = target > openSmoothed ? 0.5 : 0.2
  openSmoothed += (target - openSmoothed) * k
  return openSmoothed
}
```

